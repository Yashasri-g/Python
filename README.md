# Python
Assignments 

This repository contains two simple beginner Python programs. Below are brief descriptions of what each program does.

## 1. Basic Mathematical Operations
- Accepts two numbers from the user.
- Performs the four basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Displays the result of each operation.

## 2. Personalized Greeting
- Accepts the user's first name and last name.
- Concatenates the first and last name into a full name.
- Prints a personalized greeting message that includes the full name.
